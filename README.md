# AppToDo_Jackpack_compose

* UI completely in [Jetpack Compose]
* Uses [Kotlin Coroutines]
* Uses [Kotlin Flow]
* Uses many of the [Architecture Components]
* Uses [Hilt]
Uses [Java 11+ API desugaring support]

## Prerequisites

* Android Studio Chipmunk | 2021.2.1
* Min SDK 21
* Target SDK 33
* AGP 7.2.1
* Java 18
* Kotlin 1.7.10
